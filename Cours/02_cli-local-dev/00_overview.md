# Chapter 02 — CLI & Local Development

## What you'll learn

- Install and configure the Supabase CLI
- Run a full Supabase stack locally (PostgreSQL, Auth, Storage, Studio)
- Manage database schema changes with migrations
- Seed your local database with test data
- Link your local project to a remote Supabase project and push changes

## Prerequisites

- Chapter 01 completed (PostgreSQL fundamentals)
- Docker installed and running
- Node.js 18+ (for `npx`) or Homebrew (for `brew`)
