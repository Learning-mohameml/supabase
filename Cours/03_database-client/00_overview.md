# Chapter 03 — Database Client

## What you'll learn

- How Supabase exposes your PostgreSQL tables as a REST API (PostgREST)
- Set up the Supabase JavaScript client in a Next.js app (browser + server)
- Perform CRUD operations from the client (insert, select, update, delete)
- Filter, sort, and paginate queries
- Fetch related data across tables (joins via foreign keys)
- Generate TypeScript types from your database schema

## Prerequisites

- Chapter 02 completed (local stack running, migrations applied, seed data loaded)
- Next.js app initialized in `Projects/todos/`
- `supabase start` running locally
